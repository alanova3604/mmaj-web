# MMAJ Cocinas Integrales 🍳

A premium front-end application designed and developed for **MMAJ Cocinas Integrales**, focusing on high-performance user experience and modern visual aesthetics.

---

## 🚀 Key Features

- **Responsive Architecture:** Fully optimized for mobile, tablet, and desktop viewing.
- **Modern Tech Stack:** Built with Next.js 15 (App Router) for superior performance and SEO.
- **Fluent Animations:** Integrated smooth transitions using `Framer Motion` for an interactive feel.
- **Tailwind CSS:** Utility-first styling for a clean, maintainable, and scalable codebase.
- **Analytics Ready:** Integrated with Microsoft Clarity for user behavior insights.

---

## 🛠️ Technology Stack

- **Framework:** [Next.js](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Typography:** [Montserrat](https://fonts.google.com/specimen/Montserrat) (Google Fonts)
- **Deployment:** Vercel

## 📦 Getting Started

### 1. Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/alanova3604/mmaj-web.git
cd mmaj-web
npm install
```

### 2. Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

---

## 🏗️ Architecture Note

This project uses the **Next.js App Router** structure:

- `/src/app`: Root layouts and page definitions.
- `/src/componentes`: Reusable UI components and business logic.
- `/src/componentes/IU`: Core atomic UI elements.
- `/src/componentes/layout`: Structural components (Header, Footer).
